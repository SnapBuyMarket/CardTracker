
Alternate Startup Installer (v6.3 ALT)
--------------------------------------
This avoids creating a .lnk shortcut (which some systems block) and instead
creates a small .CMD file in your Startup folder that calls Start_All_v6.bat.

How to use:
1) Put this file next to Start_All_v6.bat (your ScanToCardTracker folder).
2) Double-click Install_Startup_ALT.bat
3) Double-click Open_Startup_Folder_ALT.bat to verify you see:
     CardTracker Start-All (v6).cmd
