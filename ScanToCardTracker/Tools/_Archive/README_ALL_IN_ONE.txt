
CardTracker — All-In-One v6.3 Pack
==================================

This pack includes:
 • v6.3 Start-All with self-finding OneDrive AutoMove
 • Startup installer that removes old v5 shortcut automatically
 • Excel Panel (UF_CardTracker.frm + modCardTracker.bas) + instructions

Quick Start
-----------
1) Extract this ZIP directly into your ScanToCardTracker folder.
2) Double-click Set_Project_Path.bat and paste your path if asked.
3) Double-click Start_All_v6.bat — you should see OCR Watcher + AutoMove Watcher (v6.3).
4) (Optional) Run Install_Startup.bat → then Open_Startup_Folder.bat (delete any old v5 link if still present).
5) Import the Excel Panel (see ExcelPanel/README_Excel.txt).

Logs
----
 • Logs\automove_watch.log — shows source folders and moves.
 • Logs\ocr_watch.log — shows OCR watcher activity.


ALT Startup included: use Install_Startup_ALT.bat if the normal installer doesn't create the shortcut.
If the UserForm import gives errors, use ExcelPanel_MacrosOnly/modCardTracker.bas instead.
